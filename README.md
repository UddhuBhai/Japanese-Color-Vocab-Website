# 🇯🇵 Japanese Vocabulary - Colors

A simple and beginner-friendly Japanese vocabulary website built using HTML and CSS. This project helps users learn basic Japanese color names through visual representation and color-coded text.

## 📖 About

This website introduces five common colors in Japanese along with their English equivalents and matching images.

| English | Japanese (Romaji) | Japanese |
|----------|------------------|-----------|
| Red | Aka | 赤 |
| Blue | Ao | 青 |
| Orange | Orenji | オレンジ |
| Green | Midori | 緑 |
| Yellow | Kiiro | 黄色 |

## ✨ Features

- Clean and minimal user interface
- Color-coded Japanese vocabulary
- Visual learning with images
- Beginner-friendly layout
- Responsive image sizing
- Built entirely with HTML5 and CSS3
- Lightweight and easy to customize
